## 概述
>PerfView 是 .NET 团队创建的一种性能分析工具，专门用于诊断 .NET 性能问题。 PerfView 允许分析 CPU 使用率、内存和 GC 行为、性能事件以及挂钟时间。
[Github相关链接](https://github.com/Microsoft/perfview?tab=readme-ov-file)
